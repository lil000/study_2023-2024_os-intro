2. Написfkb на языке Си программу, которая вводит число и определяет, является ли оно
больше нуля, меньше нуля или равно нулю. Затем программа завершается с помощью

листинг программы:
echo "Insert number:"
read n
if [ $n -gt 0 ]
then echo ">0"
elif [ $n -eq 0 ]
then echo "=0"
else echo "<0"
fi
done
3. Написали командный файл, создающий указанное число файлов, пронумерованных
последовательно от 1 до  (например 1.tmp, 2.tmp, 3.tmp,4.tmp и т.д.). Число файлов,
которые необходимо создать, передаётся в аргументы командной строки. Этот же
командный файл должен уметь удалять все созданные им файлы (если они существуют)
Листинг программы:
while getopts "c:r" opt
do
case $opt in
c)n="OPTARG; for i in $(seq 1 $n); do toch "$i.tmp"; done;;
r)for i in $(find -name "*.tmp"); do rm $i; done;;
esac
done
